# Synthetic Financial Data Generation using VAE, GAN & CTGAN

## 📌 Overview

This project presents a Generative AI framework for creating synthetic financial datasets to improve credit risk assessment while preserving privacy and reducing demographic bias.

---

## 🚀 Features

- Variational Autoencoder (VAE)
- Generative Adversarial Network (GAN)
- Conditional Tabular GAN (CTGAN)
- Synthetic Data Generation
- Credit Risk Classification
- Privacy Evaluation
- Fairness Evaluation
- Statistical Fidelity Analysis
- Interactive Streamlit Dashboard

---

## 🛠 Technologies

- Python
- Streamlit
- PyTorch
- CTGAN
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
